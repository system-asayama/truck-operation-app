📋 デバイス設定方法（API仕様）

GPSデバイスから以下のエンドポイントにHTTP POSTで位置情報を送信してください。

POST
https://truck.samurai-hub.com/api/gps_device/location

Headers:
Authorization: Bearer {APIトークン}
Content-Type: application/json

Body (JSON):
{
  "latitude": 35.6762,
  "longitude": 139.6503,
  "accuracy": 5.0,  // 任意
  "speed": 60.0,     // 任意 (km/h)
  "recorded_at": "2026-04-26T10:00:00Z" // 任意
}

※ Teltonika形式（lat/lng）、Queclink形式も自動対応しています。
※ 送信間隔は30秒〜1分を推奨します（通信量削減のため）。
